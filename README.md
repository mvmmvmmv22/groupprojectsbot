# Телеграм бот для создания совместных проектов.
Мой телеграм бот для создания проектов в группе, созданный в качестве школьного проекта. Сейчас реализован только базовый функционал. В планах функции напоминаний о дедлайнах и раздачи задач участникам.
В качеста базы данных используется PostgreSQL, в репозитории есть файл для создания бд и таблиц в ней.
